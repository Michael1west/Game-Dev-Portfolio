# GameDevPortfolio

A collection of game development systems and projects built in Unity.
Each project is self-contained and designed to be reusable across future games.

## Projects

| Project | Description | Status |
|---------|-------------|--------|
| Locomotion System | Third-person character controller with state machine architecture and ledge traversal | ✅ Feature Complete |

## Structure

- **Projects/** — Individual game systems and projects
- **SharedSystems/** — Reusable components that work across multiple projects
